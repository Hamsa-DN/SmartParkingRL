# Smart Parking Allocation System using Deep Reinforcement Learning

## SDG Connection
SDG 11 – Sustainable Cities and Communities

## Problem Statement
The project aims to reduce parking congestion and waiting time using a Deep Q-Network (DQN)-based smart parking allocation system.

## Features
- Smart parking simulator
- Deep Reinforcement Learning (DQN)
- Dynamic parking allocation
- Reward optimization
- Experiment tracking
- Performance graphs
